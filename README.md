# vibe-coding-project
# Advanced Todo List App

## Açıklama
Bu Todo List uygulaması, görev ekleme, düzenleme, silme ve tamamlandı olarak işaretleme özelliklerine sahip gelişmiş bir web uygulamasıdır. 
Görevler local storage ile tarayıcıya kaydedilir, böylece sayfa yenilense bile veriler kaybolmaz.

## Özellikler
- Görev ekleme
- Görev düzenleme
- Görev silme
- Görevleri tamamlandı olarak işaretleme
- Görevlerin eklenme tarih ve saatini gösterme
- Local storage ile veri saklama
- Responsive ve modern tasarım

## Teknolojiler
- HTML, CSS, JavaScript
- Vibe coding aracı: Cursor
